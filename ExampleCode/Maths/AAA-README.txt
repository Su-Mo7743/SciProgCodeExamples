Set of Python example programs illustrating basic mathematics and how to call
functions from the math module.

EvaluateQuadratic.py      evaluates a a*x**2 + b*c + c quadtartic

EvaluateSinCos.py	  converts angle from degrees to radians and evaluates
			  sin / cos functions.

Cosine Rule.py		  Calcuate side lengths and angles of a triangle
       			  using cosine and sine rules.

BasicComplex.py		  Basic use of complex, read complex, basic calcualtion
			  and display

MultiplyByLogs.py	  demonestrates using log / exp to perform
			  multiplication and division.

MultiplyByLogsNagativeFixed.by  same as program above but with fix to
				deal with negative numbers
				(a bit more compicated)

EvaluateInput.py		example of using eval() to evalate a string to float() ann comlplex().
				This replicates the old Phyton 2 behaviour.

ComplexImpedance.py	  Application of complex to calcualte the complex
			  inpedance of LRC circuit.
