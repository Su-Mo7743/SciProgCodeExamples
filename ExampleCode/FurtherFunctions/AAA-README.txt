This folder contains some more advanced use of functions


PointsInCircle.py    : Use of function to return two floats as a list,
		     and also some more advanced Matplotlib to display scatter plots and annonate graphs.

FunctionPlot.py	     Passing a a function as the argument to another function, also returning lists
		     from functions.

ListofFunctions.py   Show that fuctions can be held in a list and the list elements can be called as functions.
		     Show the flexibility of lists, but this is not good programming style.

FunctionsFromCommand.py	  Show functions names being entered from command line and then called.
