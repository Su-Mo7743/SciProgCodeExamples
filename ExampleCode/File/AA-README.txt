
Set of example for reading and writing to simple text file

ReadTextFile.py   	   Read a text file and print out again
			   (discussed "\n" problem with print())

WriteFile.py               Writes a bit of text (and culture) to a file.
			   (this write / overwrites "burns.txt")

RangeLooptoFile.py	   Write data to a file from a a loop.


LineCount.py		   Open a file and countv the number of lines
			   (with error trap for wrong filename)

ImageShow.py		   Read and display an image file using Matplotlib 

Data supplied

burns.txt		A bit of culture !!!

cosdata.txt		Output from RangeLooptoFile.py

sample.txt		Data for Checkpoint 4

stones.jpg		Stones on Scottish beach
